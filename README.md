## Requirements

- Golang
- Go Module
- Postgres Database

## Installing

- Use Go Modules, please read https://blog.golang.org/using-go-modules
- Install Dependecies
  ```console
  $ go mod tidy
  ```
- Fill your local .env configuration
- Run Project
  ```console
  $ go run .
  ```
